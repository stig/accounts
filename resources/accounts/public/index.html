Welcome to Duct

Congratulations! Your project accounts is ready and running.

This is a static welcome page located at resources/accounts/index.html in the project directory. Remove or replace it when you start developing. If you remove the index page entirely, be sure to change the :aliases map in accounts.system/base-config.

Template profiles used:

+cljs
Adds ClojureScript support. Try running (cljs-repl) in the REPL.
+example
Adds an example endpoint at /example.
+ragtime
Adds Ragtime migrations. Use (migrate) and (rollback) in the REPL. Migrations are stored in resources/accounts/migrations.
+site
Adds middleware and configuration suited for a user-facing website.
+sqlite
Adds a SQLite dependency and database component. The database used for development defaults to db/dev.sqlite.
